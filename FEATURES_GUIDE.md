# Features Guide

This guide covers the main tools in Aurelia Editor and how to use them.

## Productivity

### Command Palette
Press `Ctrl/Cmd + K` to open a searchable command menu.

### Slash Commands
Type `/` in the editor to insert blocks and open quick actions.

### Format Painter
Copy the formatting from selected text and apply it elsewhere.

### Autosave
Drafts are saved locally so you can recover work after a refresh or crash.

### Find and Replace
Open search from the toolbar or `Ctrl/Cmd + F`.

## Writing and Style

### Typography
Use the `Typography` panel to set:
- font family
- font size
- font color
- highlight color

### Comments
Select text and add a comment from the toolbar or command palette.
You can reply, resolve, and filter comment threads.

### Track Changes
Toggle suggestions mode to review edits before accepting them.

### Templates
Insert reusable page blocks like hero sections, callouts, FAQs, and signatures.

## Media

### Images
Use the image tools panel to edit:
- alt text
- caption
- alignment
- size
- link

### Video and HTML
You can embed YouTube videos or insert custom HTML snippets.

## Structure

### Tables
Use the toolbar table actions or the inline table popover for row and column edits.

### Code Blocks
Insert formatted code blocks with language-aware styling.

### Table of Contents
Generate a TOC from document headings.

### Footnotes
Add document notes and references when you need academic or editorial detail.

## CMS and Publishing

### CMS Panel
Manage page metadata like:
- title
- slug
- meta description
- excerpt
- canonical URL
- featured image
- noindex

### SEO Audit
Check page readiness, content depth, and metadata quality.

### Publish Workflow
Review metadata, SEO, and readiness in one place before publishing.

### Export Presets
Choose PDF or Word export settings and adjust filename, margin, and layout options.

## Collaboration

### Presence
Open the presence panel to see live same-browser collaborators.

### Revisions
Use the revision history and track changes tools for review flows.

## Short Version

If you are editing content, the main flow is:
1. Write the page.
2. Add images, tables, or templates.
3. Open CMS and SEO.
4. Review Publish.
5. Export or copy the final HTML.
