This guide details how to use the advanced features available in **Aurelia Editor**.

## üìù Productivity Essentials

### ‚ö° Slash Commands
Access tools without leaving the keyboard.
1. Type `/` on a new line.
2. A menu will appear with options (e.g., "Heading 1", "Image", "Table").
3. Use `Arrow Keys` to navigate and `Enter` to select.

### üé® Format Painter
Copy formatting from one section to another.
1. Select text with the desired style (e.g., Bold Red).
2. Click the **Paint Roller** icon in the toolbar.
3. Select the target text to apply the style.
**Pro Tip**: Double-click the icon to lock it and apply styles multiple times. Press `Esc` to unlock.

### üíæ Autosave
The editor creates a local backup of your content automatically.
- If you accidentally close the tab, reloading the page will prompt you to **Restore Draft**.
- Autosave triggers 3 seconds after you stop typing.

### üîç Find & Replace
1. Press `Ctrl + F` (or `Cmd + F` on Mac).
2. Type your search term. Matches are highlighted.
3. Use arrows to navigate matches.
4. Expand the panel to see **Replace** options (Replace One / Replace All).

## üñºÔ∏è Media Management

### Inserting Images
- **Drag & Drop**: Drag an image file explicitly into the editor.
- **Paste**: Copy an image and paste (`Ctrl + V`) directly.
- **Toolbar**: Click the **Image** icon to upload or enter a URL.

### Managing Images
Click an inserted image to reveal the **Image Toolbar**:
- **Resize**: Drag the corner handles.
- **Link**: Add a hyperlink to the image.
- **Caption**: Add a text caption below the image.

### Video Embedding
1. Click the **Video** icon (or type `/video`).
2. Paste a YouTube URL (e.g., `https://www.youtube.com/watch?v=...`).
3. The editor automatically embeds the video player.

## üìä Tables & Data

### Creating Tables
1. Click the **Table** icon.
2. Use the grid picker to select dimensions (e.g., 3x3).
3. The table is inserted.

### Editing Tables
Hover over the top or left borders of the table to see control handles:
- **Right-Click** inside a cell for context menu:
  - Insert Row Above/Below
  - Insert Column Left/Right
  - Delete Row/Column/Table
  - Merge Cells

## üèóÔ∏è Structure & Layout

### Table of Contents (TOC)
automatically generate a list of clickable links based on your headings.
1. Ensure your document has Headings (`H1`, `H2`, etc.).
2. Place cursor where you want the TOC (usually top).
3. Click the **TOC** icon.
4. The TOC updates automatically as you edit headings.

### Footnotes
Add academic references or notes.
1. Place cursor at the reference point.
2. Click the **Footnote** icon.
3. The editor jumps to the footer where you can type the note content.

### Page Breaks
Force content to start on a new page (useful for PDF export).
- Click the **Page Break** icon.
- A visual divider will appear.

---

## üîÑ Collaboration & Review

### Track Changes
Toggle **Suggestion Mode** to propose edits without changing the original text.
1. Click the **Eye/Track Changes** icon in the toolbar.
2. Type or delete text. Changes appear as green (added) or red (deleted) highlights.
3. Click a change to **Accept** (Checkmark) or **Reject** (X) it.

### Zen Mode (Focus)
Eliminate distractions.
- Click the **Zen Mode** icon (Expand arrows).
- The toolbar and sidebar fade away, leaving only the canvas.
- Press `Esc` or click the toggle to exit.

---

## üì§ Exporting

### PDF Export
1. Click the **PDF** icon.
2. The editor generates a high-quality PDF, respecting page breaks and styles.
3. The file downloads automatically.

### Word Export/Import
- **Export**: Click the **Word** icon to download `.docx`.
- **Import**: Click **Import** and select a `.docx` file. The editor converts it to HTML.
